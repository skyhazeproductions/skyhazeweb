A project by Andrei Zagorschi

Skyhaze Productions is an indie game initiative focused on small, polished experiences built by one person.

Final Defenders: A solo-developed mobile game combining strategy, tower-defense, and clicker type elements.

What makes it stand out? The game never ends! You will continuously buy upgrades suited to your play style, and the enemy will keep getting stronger!

It also features multiple ways to make sure you never get stuck at one specific level.
